# Product Images

Drop your images here. The site will automatically show them on product cards.
If an image is missing, the card falls back to the emoji — nothing breaks.

## Required filenames

| File | Product |
|------|---------|
| `saarthi-plus.jpg` | Saarthi Plus Passenger Rickshaw |
| `cargo-king.jpg` | Cargo King Loader Rickshaw |
| `city-rider.jpg` | City Rider Standard |
| `volt-x1.jpg` | Volt X1 Pro Scooter |
| `swift-mini.jpg` | Swift Mini Scooter |
| `speed-pro.jpg` | Speed Pro Performance |
| `battery-60v.jpg` | Lithium-Ion Battery Pack 60V |
| `bldc-motor.jpg` | BLDC Hub Motor 1000W |
| `smart-charger.jpg` | Smart Charger 60V/72V |

## For videos

Add a `videos/` folder at the same level and drop MP4 files there.
Name them `hero.mp4` for a homepage video hero, or `product-<name>.mp4` for product detail videos.

## Tips
- Recommended size: 800×600px minimum, 16:9 or 4:3 ratio for product cards
- JPG works best for photos (smaller file size)
- WebP is even better for performance if your images support it
